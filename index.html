🗺️ 토토맵

전국 스포츠토토 판매점을 한눈에

앱 소개

토토맵은 전국의 스포츠토토 판매점을 쉽고 빠르게 찾을 수 있도록 도와주는 안드로이드 애플리케이션입니다. 지도 기반 인터페이스로 현재 위치 주변의 판매점을 한눈에 확인할 수 있습니다.

주요 기능

  • 네이버 지도 기반의 직관적인 판매점 위치 표시
  • 현재 위치 주변 판매점 자동 검색
  • 판매점 상세 정보 제공 (주소, 전화번호 등)
  • 판매점까지의 길찾기 기능
  • 즐겨찾기 기능으로 자주 방문하는 판매점 관리
  • 정기적인 판매점 정보 업데이트
